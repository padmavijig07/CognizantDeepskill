# Week V Examples

Frontend examples inspired by the Week 5 topics in the referenced repository.

## Programs

- `Responsive Page using Flexbox & Grid.html`: responsive dashboard layout using Flexbox and CSS Grid.
- `React SPA (useState + useEffect + React Router).html`: React single-page task board using state, effects, and hash-based routing.
- `Vue 3 Reactive List & Filter UI.html`: Vue 3 reactive product list with search and category filtering.
- `Angular (CLI Project + Data Binding + Routing)/`: Angular-style standalone component example showing interpolation, two-way binding, event binding, and routing concepts.

The HTML examples can be opened directly in a browser with an internet connection because they load React, Vue, and supporting libraries from CDNs.
