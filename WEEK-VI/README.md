# Week VI Examples

Examples inspired by the Week 6 topics in the referenced repository.

## Programs

- `Selenium Browser Automation Script in Python.py`: opens a page, verifies its title, and captures a screenshot.
- `Selenium POM (Page Object Model - 2 Pages).py`: demonstrates a two-page Page Object Model with a login page and dashboard page.
- `Code Review (3 Structured PR Comments).md`: sample structured pull-request review comments.
- `Git Feature Branch PR and Merge Conflict.txt`: practice workflow for a feature branch, pull request, and conflict resolution.

Install Selenium before running the browser examples:

```text
pip install selenium
python "Selenium Browser Automation Script in Python.py"
```

Recent Selenium versions can manage the browser driver automatically. A Chrome or Firefox installation is still required.
